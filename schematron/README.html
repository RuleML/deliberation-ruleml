This directory contains *.sch Schematron files for validating constraints on psoa expressions in *.ruleml instance files as found, e.g., in http://deliberation.ruleml.org/1.03-psoa/exa/. PsoaExpressionConstraints.sch will report psoa-expression constraint violations of candidate instances with or without an xmlns attribute indicating the ruleml namespace (i.e., <RuleML xmlns="http://ruleml.org/spec" ...>). Two stand-alone Schematron files are also available: PsoaExpressionConstraintsNS.sch, only for candidate instances with the ruleml namespace, and PsoaExpressionConstraintsNoNS.sch, only for candidate instances without the ruleml namespace.
An instance intended to fail, naffologeqPSOA_fail.ruleml, is also included.
