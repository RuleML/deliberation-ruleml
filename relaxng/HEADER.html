RuleML 1.0 RelaxNG Directory

Notice of Pending Obsolescence

This directory contains the approved Relax NG schemas of Deliberation RuleML. The Specification of Deliberation RuleML 1.01is currently under review. In the pending release, the Relax NG schemas are available from the Deliberation RuleML 1.01 Relax NG Directory and also from MYNG.

This directory contains the "modules/" subdirectory of core modules and the "modules_ext/" subdirectory of extension modules. Also a few static driver files are published here for demonstration purposes.
